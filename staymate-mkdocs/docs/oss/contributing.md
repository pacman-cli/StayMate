# Contributing

How to contribute to StayMate.

---

## Getting Started

1. **Fork** the repository
2. **Clone** your fork
3. **Create** a feature branch
4. **Implement** your changes
5. **Test** thoroughly
6. **Submit** a Pull Request

---

## Development Setup

```bash
# Clone
git clone https://github.com/YOUR_USERNAME/staymate.git
cd staymate/server

# Run
./mvnw spring-boot:run
```

---

## Pull Request Guidelines

- [ ] Code follows [Code Style](code-style.md)
- [ ] Tests pass (`./mvnw test`)
- [ ] New features have tests
- [ ] Documentation updated

---

## Commit Messages

Use conventional commits:

```
feat: add new booking feature
fix: resolve token expiry issue
docs: update API documentation
test: add property service tests
```
