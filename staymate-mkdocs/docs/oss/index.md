## Quick Start

```bash
# Fork and clone
git clone https://github.com/YOUR_USERNAME/staymate.git

# Create branch
git checkout -b feature/your-feature

# Make changes, commit, push
git commit -m "feat: add new feature"
git push origin feature/your-feature

# Open Pull Request
```
